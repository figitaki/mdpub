# Checklist Demo

This page demonstrates GitHub-style task list support.

## Simple Task List

- [ ] Buy groceries
- [x] Write documentation
- [X] Fix the bug
- [ ] Deploy to production

## Mixed List

Here's a list with both tasks and regular items:

- [ ] Unchecked task
- [x] Checked task
- Regular list item (not a task)
- Another regular item

## Nested Task Lists

- [ ] Parent task
  - [ ] Child task 1
  - [x] Child task 2
- [x] Completed parent
  - [x] All children done
  - [x] Really done
